Mysql FIND_IN_SET()

发表于 2018-10-08 | 分类于 Mysql随笔 |

FIND_IN_SET应用情景

如：
在商品表中，有个城市字段存储哪几个城市有货，如广州、汕头、珠海，存储的值是200,754,756
如果要查询汕头有货的商品，不能用like，需要用到FIND_IN_SET()

FIND_IN_SET()语法

FIND_IN_SET(str,strlist)

like是广泛的模糊匹配，字符串中没有分隔符，find_in_set是精确匹配，字段值以英文”,”分隔
